## サービス概要
仏教みつけは、1日1回、日常の中の出来事を「仏教の視点」で見つめ直せる投稿サービスです。  
与えられたお題（仏教の教え）に当てはまる瞬間を思い出し、短い文章として投稿できます。  
知識がなくても気軽に仏教の智慧に触れられ、他の人の視点を通して新しい気づきを得られます。

## サービスURL

https://buddhalog.com/

## サービスの開発背景
現代では、仏教が「学問」や「葬儀の文化」として距離を置かれがちですが、本来仏教は日常に根ざした生き方の気づきを与えるものです。
私自身も、以前は「難しくて形式ばったもの」という印象を持っていました。
しかし、学んでいくうちに、仏教は生きるうえで当たり前に起きていることを言葉で丁寧に捉えた「人間理解の知恵」なのだと知り、深く惹かれました。

仏教を学んで以来、身近な人の悩みを聞く中で「この人が仏教の視点を少しでも持てたら悩みや迷いを軽くできるかもしれない」と感じることがよくあり、それをきっかけに、誰でも手軽に仏教を感じられるサービスを作ろうと思いました。

仏教に関する既存のアプリには、教義や経典を学ぶ“座学型”か、瞑想や呼吸法を中心とした“実践型”のどちらかに偏ったものが多く、その両方をバランスよく、手軽に日常で取り入れられるサービスは見当たりませんでした。

このアプリはそれら既存の専門的なサービスほど深い学びにはならないかもしれませんが、仏教の入り口として最適なアプリとなっています。

## 機能紹介

### ・ユーザー登録/ログイン
|<img width="1258" height="766" alt="スクリーンショット 2026-01-06 151628" src="https://github.com/user-attachments/assets/81217ecc-0652-4026-a13f-e4a25231bdc7" />|
|:-|

名前、メールアドレス、パスワード、確認用パスワードを入力してユーザー登録を行います。
ユーザー登録後はマイページへ遷移するためすぐにサービスを利用可能です。

また、Googleアカウントを用いてログインすることもできます。

### ・仏教をみつける
|<img width="2540" height="1303" alt="スクリーンショット 2026-01-06 224909" src="https://github.com/user-attachments/assets/cdcb47b6-4d76-4be3-85b5-42f0785d3a00" />|

|<img width="2541" height="1303" alt="スクリーンショット 2026-01-06 224938" src="https://github.com/user-attachments/assets/882dcc5f-6388-4886-8875-bd2a45c7df84" />|

マイページの「今日の仏教をみつける」ボタンを押すとお題と投稿フォームが表示されます。
お題に当てはまる内容を日常生活の場面からみつけて送信ボタンを押すと投稿完了です。

投稿には画像も添付可能です。

### ・マイページの機能
|<img width="1262" height="1302" alt="スクリーンショット 2026-01-06 230748" src="https://github.com/user-attachments/assets/5860de18-660f-4287-9ccc-b8e2cbcabb67" />|

投稿完了後は、マイページのカレンダーに投稿した日と投稿されたお題が表示されます。

カレンダーをクリックすることで投稿の詳細を表示することが出来ます。

### その他機能
- 他ユーザーの投稿を見られる投稿の一覧表示（カードをクリックで投稿の詳細へと移動可能）
- 他ユーザーの投稿へリアクション
  - リアクションした投稿のリスト化
- 自分の投稿履歴表示
- おすすめの仏教入門書籍紹介（楽天books）

## 仕様技術
- バックエンド: Ruby on Rails 7.2, Ruby 3.3
- フロントエンド: Ruby on Rails
- CSSフレームワーク: Tailwindcss
- Web API: Rakuten API, Google API
- ホスティングサービス: Render.com
- データベースサーバー: PostgreSQL(Neon)
- ストレージサーバー: Cloudinary
- バージョン管理: GitHub
